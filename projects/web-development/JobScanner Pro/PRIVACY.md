# Privacy Policy for JobScanner Pro

## Overview
JobScanner Pro is a job search aggregator that helps users find creative opportunities across Canada. This privacy policy outlines how we handle data and protect user privacy.

## Data Collection
JobScanner Pro collects and processes the following types of data:

### Job Listing Data
- Public job postings from various sources (Job Bank, Kijiji, Craigslist)
- Basic job information (title, company, location, description)
- URLs to original job postings
- Posting dates and other public metadata

### User Data (Optional)
If you choose to use the auto-apply feature (coming soon), we may collect:
- Name and contact information
- Resume and portfolio links
- Job application history
- Search preferences and filters

## Data Storage
- All data is stored locally on your device
- No data is transmitted to external servers except when:
  - Searching job boards
  - Applying to jobs (with your explicit consent)
  - Checking for updates

## Third-Party Services
We interact with the following third-party services:
- Job Bank (jobbank.gc.ca)
- Kijiji (kijiji.ca)
- Craigslist (toronto.craigslist.org)
- Indeed.ca (when available)

These interactions are limited to:
- Searching for job listings
- Retrieving public job information
- Submitting applications (with your consent)

## Data Protection
We implement several measures to protect your data:
- Local storage only - no cloud sync
- No tracking or analytics
- No sharing of personal information
- Secure handling of credentials

## Your Rights
You have the right to:
- Access your stored data
- Delete your data
- Opt out of auto-apply features
- Control what information is saved

## Updates to Privacy Policy
We will notify users of any changes to this privacy policy through:
- Application notifications
- GitHub repository updates
- Release notes

## Contact
For privacy-related questions or concerns:
- Open an issue on GitHub
- Contact the maintainer directly
- Check the README for additional contact methods

## Compliance
JobScanner Pro is designed to comply with:
- Canadian Privacy Laws
- PIPEDA Guidelines
- Terms of service of job boards
