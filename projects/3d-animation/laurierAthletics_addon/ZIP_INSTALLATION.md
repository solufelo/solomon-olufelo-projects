**Solutions:**

- Don't extract the zip file - install it directly

**Problem:** Addon is enabled but panel doesn't appear

**Solutions:**
- Restart Blender completely
- Check that sidebar is open (press N)

- Ensure you're in 3D Viewport workspace

**Problem:** Error when creating animation

- Check Blender console for error messages
