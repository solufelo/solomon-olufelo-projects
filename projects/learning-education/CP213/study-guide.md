## üìù Study Techniques

### **Active Learning Strategies**
1. **Code Along**: Type every example from the textbook
2. **Modify Examples**: Change parameters and see what happens
3. **Create Variations**: Write similar programs with different data
4. **Debug Intentionally**: Introduce bugs and practice fixing them
5. **Explain Concepts**: Teach someone else what you've learned

### **Flashcard Strategy**
1. **Daily Review**: Spend 15-30 minutes daily reviewing flashcards
2. **Spaced Repetition**: Use Anki or similar for optimal retention
3. **Active Recall**: Try to answer before looking at the answer
4. **Mix Topics**: Don't study only one topic at a time
5. **Regular Testing**: Take practice quizzes weekly

### **Programming Practice**
1. **Start Simple**: Begin with basic programs and build complexity
2. **Read Error Messages**: Learn to interpret compiler errors
3. **Use Debugger**: Practice with breakpoints and variable inspection
4. **Comment Code**: Write clear comments explaining your logic
5. **Test Thoroughly**: Test with various inputs and edge cases
