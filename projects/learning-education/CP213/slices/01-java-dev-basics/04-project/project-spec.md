Your program must:

## ğŸ’¡ Hints

1. **For alignment**: Use `System.out.printf("%-20s %6.2f\n", label, value)`
   - `%-20s` = left-align string in 20 chars
   - `%6.2f` = right-align number, 6 chars wide, 2 decimals

2. **For borders**: Use repeated characters
   ```java
   System.out.println("=".repeat(37));
   ```

3. **For method return**: Don't forget `return` statement
   ```java
   public static double calculateAverage(...) {
       double avg = ...;
       return avg;  // Don't forget this!
   }
   ```

4. **For letter grades**: Use if-else chain
   ```java
   if (avg >= 90) return "A";
   else if (avg >= 80) return "B";
   // ... etc
   ```
