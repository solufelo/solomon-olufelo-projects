### **Chapter 1: Getting Started**
- [ ] **Java Basics**
  - [ ] Understand Java history and origins
  - [ ] Know Java's key features (portable, object-oriented, secure)
  - [ ] Understand bytecode and JVM concepts
- [ ] **Program Structure**
  - [ ] Know basic program structure (class, main method)
  - [ ] Understand compilation and execution process
  - [ ] Know how to run Java programs
- [ ] **Variables and Expressions**
  - [ ] Know primitive data types (int, double, char, boolean)
  - [ ] Understand variable declaration and initialization
  - [ ] Know arithmetic operators (+, -, *, /, %)
  - [ ] Understand operator precedence
- [ ] **String Class**
  - [ ] Know String object creation
  - [ ] Understand string concatenation with +
  - [ ] Know basic string methods (length, charAt, substring)
- [ ] **Program Style**
  - [ ] Know naming conventions for variables and classes
  - [ ] Understand constant declaration with final
  - [ ] Know proper commenting practices

### **Chapter 2: Console Input and Output**
- [ ] **Screen Output**
  - [ ] Know difference between print and println
  - [ ] Understand string concatenation in output
  - [ ] Know escape sequences (\n, \t, \\, \")
- [ ] **Console Input**
  - [ ] Know how to create Scanner object
  - [ ] Understand nextInt(), nextDouble(), nextLine() methods
  - [ ] Know how to handle nextInt()/nextLine() buffer issue
- [ ] **File Input**
  - [ ] Know FileInputStream and FileNotFoundException
  - [ ] Understand try-catch blocks for file operations
  - [ ] Know hasNextLine() method for end-of-file detection
  - [ ] Understand file closing with close() method

### **Chapter 3: Flow of Control**
- [ ] **Branching**
  - [ ] Know if-else statement syntax
  - [ ] Understand Boolean expressions
  - [ ] Know comparison operators (==, !=, <, >, <=, >=)
  - [ ] Understand string comparison with equals() method
- [ ] **Boolean Logic**
  - [ ] Know logical operators (&&, ||, !)
  - [ ] Understand short-circuit evaluation
  - [ ] Know operator precedence
- [ ] **Loops**
  - [ ] Know while loop syntax and use
  - [ ] Understand do-while loop (executes at least once)
  - [ ] Know for loop syntax and common patterns
  - [ ] Understand loop control (break, continue)
- [ ] **Debugging**
  - [ ] Know common loop errors (infinite loops, off-by-one)
  - [ ] Understand variable tracing techniques
  - [ ] Know debugging strategies
- [ ] **Random Numbers**
  - [ ] Know Random class and nextInt() method
  - [ ] Understand Math.random() method
  - [ ] Know how to generate numbers in specific ranges

### **Chapter 4: Defining Classes I**
- [ ] **Class Definitions**
  - [ ] Understand class vs object concepts
  - [ ] Know instance variable declaration
  - [ ] Understand method definition syntax
  - [ ] Know object creation with new operator
- [ ] **Encapsulation**
  - [ ] Know public vs private modifiers
  - [ ] Understand information hiding principles
  - [ ] Know API vs implementation concepts
  - [ ] Understand accessor and mutator methods
- [ ] **Method Overloading**
  - [ ] Know method signature concept
  - [ ] Understand parameter differences (type, number, order)
  - [ ] Know constructor overloading
  - [ ] Understand method resolution by compiler
- [ ] **Constructors**
  - [ ] Know constructor syntax and purpose
  - [ ] Understand default constructor
  - [ ] Know parameterized constructors
  - [ ] Understand this keyword in constructors
- [ ] **Static Members**
  - [ ] Know static method syntax and use
  - [ ] Understand static vs instance methods
  - [ ] Know static variable concepts
  - [ ] Understand when to use static methods

### **Chapter 5: Defining Classes II**
- [ ] **References**
  - [ ] Understand reference vs value semantics
  - [ ] Know how objects are stored in memory
  - [ ] Understand assignment of reference types
  - [ ] Know == vs equals() for objects
- [ ] **Privacy Leaks**
  - [ ] Understand privacy leak concept
  - [ ] Know defensive copying techniques
  - [ ] Understand getter method best practices
  - [ ] Know class invariant concepts
- [ ] **Packages**
  - [ ] Know package declaration syntax
  - [ ] Understand import statements
  - [ ] Know java.lang package (automatically imported)
  - [ ] Understand CLASSPATH environment variable

### **Chapter 6: Arrays**
- [ ] **Array Basics**
  - [ ] Know array declaration syntax
  - [ ] Understand array creation with new
  - [ ] Know indexed variable access
  - [ ] Understand array length property
- [ ] **Array Programming**
  - [ ] Know partially filled arrays
  - [ ] Understand array parameters
  - [ ] Know common array algorithms (search, sort, sum)
  - [ ] Understand array bounds checking
- [ ] **Multidimensional Arrays**
  - [ ] Know 2D array declaration and creation
  - [ ] Understand array of arrays concept
  - [ ] Know how to access 2D array elements
  - [ ] Understand ragged arrays

## üéØ Exam Strategies

### **Before the Exam**
- [ ] **Review Flashcards**: Go through all Quizlet sets
- [ ] **Practice Coding**: Write programs without looking at notes
- [ ] **Take Practice Exams**: Time yourself
- [ ] **Identify Weak Areas**: Focus extra study time
- [ ] **Get Good Sleep**: Don't stay up late studying

### **During the Exam**
- [ ] **Read All Questions First**: Understand what's being asked
- [ ] **Manage Time**: Don't spend too long on one question
- [ ] **Start with Easy Questions**: Build confidence
- [ ] **Show Your Work**: Partial credit is possible
- [ ] **Check Your Code**: Look for syntax errors
- [ ] **Test Your Logic**: Trace through your code

### **Common Mistakes to Avoid**
- [ ] **Syntax Errors**: Missing semicolons, braces, parentheses
- [ ] **Variable Declaration**: Forgetting to declare variables
- [ ] **Array Bounds**: Accessing invalid array indices
- [ ] **String Comparison**: Using == instead of equals()
- [ ] **Loop Errors**: Infinite loops or off-by-one errors
- [ ] **Null Pointer**: Accessing methods on null objects
