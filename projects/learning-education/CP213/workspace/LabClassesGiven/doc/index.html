Project Overview

This lab implements a complete Student class with proper object-oriented programming principles including encapsulation, constructors, getters, setters, and the Comparable interface.

Class Structure

Student Class Attributes

  • private LocalDate birthDate - Student's birth date
  • private String forename - Student's first name
  • private int id - Student's unique ID number
  • private String surname - Student's last name

Implemented Methods

Constructor

public Student(int id, String surname, String forename, LocalDate birthDate)

Creates a new Student object with the provided parameters.

toString() Method

public String toString()

Returns a formatted string representation of the student with all attributes.

compareTo() Method

public int compareTo(final Student target)

Implements the Comparable interface. Compares students by surname first, then by forename if surnames are equal. Returns negative, zero, or positive integer based on alphabetical ordering.

Getter Methods

  • getBirthDate() - Returns the student's birth date
  • getForename() - Returns the student's first name
  • getId() - Returns the student's ID
  • getSurname() - Returns the student's last name

Setter Methods

  • setBirthDate(LocalDate birthDate) - Sets the student's birth date
  • setForename(String forename) - Sets the student's first name
  • setId(int id) - Sets the student's ID
  • setSurname(String surname) - Sets the student's last name

Main Class Testing

The Main class provides comprehensive testing of all Student class functionality:

  • Student object creation and display
  • Testing all getter methods
  • Testing all setter methods with new values
  • Testing the compareTo method with multiple student objects

Key Programming Concepts Demonstrated

  • Encapsulation: Private attributes with public getter/setter methods
  • Constructor: Proper object initialization
  • Method Overriding: toString() and compareTo() methods
  • Interfaces: Implementation of Comparable<Student>
  • JavaDoc Documentation: Comprehensive method documentation
  • LocalDate Usage: Proper handling of date objects
