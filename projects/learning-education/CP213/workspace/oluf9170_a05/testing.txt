Testing OrderPanel (TakeOutGUI)
----------------------------------------
The OrderPanel GUI was tested by running TakeOutGUI.java.
The GUI correctly:
- Displays all menu items with names and prices
- Shows quantity text fields for each item
- Updates order when quantity fields lose focus (tab or click away)
- Validates input: non-numeric or negative values are reset to 0
- Updates subtotal, tax, and total labels in real-time
- Print button opens print dialog and prints order to PDF/printer
- Layout uses GridLayout and dynamically adjusts to menu size

- Tax row: "Taxes:" | tax value | empty
- Total row: "Total:" | total value | empty
- Print button row: Print button | empty | empty

All functionality works as specified in the assignment requirements.
