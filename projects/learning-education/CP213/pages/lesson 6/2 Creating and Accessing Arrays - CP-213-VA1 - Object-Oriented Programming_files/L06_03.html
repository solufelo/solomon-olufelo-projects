2 Creating and Accessing Arrays

An array that behaves like a collection of variables, e.g., all of type double, can be created using one statement as follows: 


	double[] score = new double[5];

Or using two statements:


	double[] score;
	score = new double[5];

The first statement declares the variable score to be of the array type double[].

The second statement creates an array with five numbered variables of type double and makes the variable score a name for the array.

Like declarations for variables, an array declaration has three components:

  • The access modifier
  • the array type
  • the array name

If an access modifier is not declared, the variable takes the default access modifier. Brackets can be put before or after the array's name:

float anArrayOfFloats[]; or float [] anArrayOfFloats;

The second one is preferable and looks more natural to declare variables; similar to declaring int x ;  you declare float [] x ;

New operator is used for creating arrays. The following line of code allocates an array with enough memory for 10 integer elements and assigns the array to the myArray variable.


     myArray = new int[10]

The individual variables that together make up the array are called indexed variables. They can also be called subscripted variables or elements of the array. The number in square brackets is called an index or subscript.

In Java, indices must be numbered starting with 0, and nothing else.

The number of indexed variables in an array is called the length or size of the array. When an array is created, the length of the array is given in square brackets after the array type. The indexed variables are then numbered starting with 0, and ending with the integer that is one less than the length of the array. For example, array of five scores will have subscripts from 0 to 4, score[0], score[1], score[2], score[3], score[4].

2.1 Array Size

The length of an array is set when the array is created. After creation, an array’s length is fixed, holds a fixed number of values. In the example below, the array size is 5.


	double[] score = new double[5];

A variable may be used in place of the integer (i.e., in place of the integer 5 above). The value of this variable can then be read from the keyboard. This enables the size of the array to be determined when the program is run. For example, the statement above can be changed to 


	double[] score = new double[count];

An array is considered to be an object. Since other objects can have instance variables, so can arrays. Every array has exactly one instance variable named length. When an array is created, the instance variable length is automatically set equal to its size.  The value of length cannot be changed (other than by creating an entirely new array with new).

2.2 Array type

An array is a homogeneous container object, i.e., all the elements of an array are of the same type, called the base type of the array. It can have indexed variables or elements of any type, including any class type.

An array is declared and created in almost the same way that objects are declared and created:

BaseType[] ArrayName = new BaseType[size];

The size may be given as an expression that evaluates to a non-negative integer, for example, an int variable. Arrays of three different types, char, double and Person, are given below:


	char[] line = new char[80];
	double[] reading = new double[count];
	Person[] specimen = new Person[100];

2.3  Referring to Arrays and Array Elements

Each array element can be used just like any other single variable by referring to it using an indexed expression:  score[0].  The array itself (i.e., the entire collection of indexed variables) can be referred to using the array name (without any square brackets):  score.

An array index can be computed when a program is run. It may be represented by a variable,  score[index] or represented by an expression that evaluates to a suitable integer:

score[next + 1]

2.4 Using Arrays in a Program

The for loop is ideally suited for performing array manipulations. See the example below: 


		double[] score = new double[5];
		int index;
		double max;

		System.out.println("Enter 5 scores:");
		score[0] = keyboard.nextDouble();
		max = score[0];

		for (index = 1; index < 5; index++) {
			score[index] = keyboard.nextDouble(); // values inserted in array
			if (score[index] > max)
				max = score[index];
	            // max is the largest of the values score[0],...,  score[index].
		}

You may have noticed that the square brackets []  in the example above is used in three different ways:

  1. To create a type name: double[] score;
  2. To create a new array: score = new double[5];
  3. To name an indexed variable of an array: max = score[0];

2.5 Non-Graded Activity

A Java program for creating and using is listed below.

Compile and run the program below and observe the results.

Make changes to the program and re-run it as many times as you want to practice using Arrays in Java. 


package lesson06;

import java.util.Scanner;

public class ExamScores {
/**
* Reads in 5 scores and shows how much each score differs 
      from the highest score.
*/
	public static void main(String[] args) {
		Scanner keyboard = new Scanner(System.in);
		double[] score = new double[5];
		int index;
		double max;

		System.out.println("Enter 5 scores:");
		score[0] = keyboard.nextDouble();
		max = score[0];
		
		for (index = 1; index < 5; index++) {
			score[index] = keyboard.nextDouble();
			if (score[index] > max)
				max = score[index];
	// max is the largest of the values score[0],..., score[index].
		}

		System.out.println("The highest score is " + max);
		System.out.println("The scores are:");
		for (index = 0; index < 5; index++)
			System.out.println(score[index] + 
					" differs from max by " + (max –
                                                   score[index]));
	}
}

2.6 Array Index Out of Bounds Error

Array indices always start with 0 and always end with the integer that is one less than the size of the array. The most common programming error made when using arrays is attempting to use a nonexistent array index. When an index expression evaluates to some value other than those allowed by the array declaration, the index is said to be out of bounds.

An out of bounds index will cause a program to terminate with a run-time error message. Array indices get out of bounds most commonly at the first or last iteration of a loop that processes the array. Be sure to test for this!

2.7  Initializing Arrays

An array can be initialized when it is declared. Values for the indexed variables are enclosed in braces and separated by commas. The array size is automatically set to the number of values in the braces:


	int[] age = {2, 12, 1};

Given age above, age.length has a value of 3.

Another way of initializing an array is by using a for loop. See the example below: 


	double[] reading = new double[100];

	int index;
	for (index = 0;  index < reading.length; index++)
		reading[index] = 42.0;

If the elements of an array are not initialized explicitly, they will automatically be initialized to the default value for their base type.

2.8  Array of Characters and String

An array of characters is conceptually a list of characters, and so is conceptually like a string.

However, an array of characters is not an object of the class String.


	char[] a = {'A', 'B', 'C'};
	String s = a; //Illegal!

However, an array of characters can be converted to an object of the type String. The class String has a constructor that has a single parameter of type char[]. 


	String s = new String(a);

The object s will have the same sequence of characters as the entire array a ("ABC") but is an independent copy.

Another String constructor uses a subrange of a character array instead


	String s2 = new String(a,0,2);

Given an array a as before, the new string object is "AB".

An array of characters does have some things in common with String objects. For example, an array of characters can be outputted using the println method: 


	System.out.println(a);

Given an array a as before, this would produce the output ABC.
