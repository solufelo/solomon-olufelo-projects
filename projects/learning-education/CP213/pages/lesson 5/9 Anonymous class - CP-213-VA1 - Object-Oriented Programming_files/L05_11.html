9 Anonymous class

The new operator invokes a constructor that initializes an object and returns a reference to the location in the memory of the newly created object. This reference can be assigned to a variable of the object's class type. Sometimes, the object created is used as an argument to a method and never used again. In this case, the object doesn't need to be assigned to a variable, i.e., given a name. An object whose reference is not assigned to a variable is called an anonymous object.

Anonymous class expression allows you to declare and instantiate a class at the same time. They are similar to local classes except that they do not have a name. This way you can keep your code more concise. In the previous example on the static variable, i.e.,  StaticWNonStatic, we used an anonymous object in the main method to invoke the print method with the anonymous object argument as follows:


	public static void main (String args[] ) {		
	 	new StaticWNonStatic().nonStaticPrint(new Object()); 
	}
